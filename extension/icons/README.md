# Extension Icons

Drop the rendered PNG icons here before publishing the extension:

- `icon16.png` (16×16) — toolbar
- `icon48.png` (48×48) — extensions page
- `icon128.png` (128×128) — Chrome Web Store / detail page

Source SVGs are in `../../docs/`:
- `AutoClock_Icon.svg` (or `.png`)
- `AutoClock_AppIcon.svg` (or `.png`)

Use the AutoClock brand mark only — no third-party imagery.
