AutoClock - Design delivery
============================

This folder contains every page of the AutoClock design as single self-contained
HTML files. Just double-click any one to open it in Chrome. No internet needed,
no companion files needed - CSS, JS, fonts, brand SVG are all inlined.

Start here:
  index.html  -  links to all 17 pages
